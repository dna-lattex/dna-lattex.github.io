### WPLicense | All Rights Reserved

Automatic updates for premium WordPress themes and plugins from your WordPress dashboard.

== Installation ==

1. Upload the entire 'wp-license-upgrades' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
